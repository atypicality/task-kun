# Tech Stack 🛠️

Task-kun AI is built with a modern, scalable stack designed for performance, flexibility, and AI integration.


## 🌱 Architecture Overview

| Layer        | Technology                          | Justification |
|-------------|--------------------------------------|--------------|
| Frontend    | React + TypeScript + Tailwind CSS    | Component-based architecture with type safety, fast development, and responsive UI design. |
| Backend     | Node.js + Express + TypeScript       | Lightweight, scalable server architecture with strong ecosystem support and API flexibility. |
| Database    | MongoDB                              | Flexible NoSQL database ideal for dynamic task, user, and activity data. |
| AI / LLM    | Gemini API                           | Cost-effective LLM with strong reasoning capabilities for task planning and prioritization. |
| TTS / Voice | Qwen Voice Model + ElevenLabs        | AI-powered voice synthesis for intelligent and engaging voice assistant interactions. |
| Deployment  | Docker + Vercel                      | Containerized environments for consistency and streamlined cloud deployment. |
---

## 🎯 Why This Stack?

- **TypeScript** improves reliability and maintainability across frontend and backend.
- **React** enables modular, scalable UI development.
- **MongoDB** supports flexible task schemas and evolving user data.
- **Gemini API** powers agentic reasoning for intelligent task decomposition.
- **Qwen Voice Model + ElevenLabs** provide high-quality AI voice interaction.
- **Docker + Vercel** ensure smooth deployment and reproducible environments.

This stack allows Task-kun AI to scale while maintaining strong AI integration and responsive performance.
